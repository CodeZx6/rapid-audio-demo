Overview

LSM and ESM are two reproducible noisy-speech benchmarks built to isolate a single distribution shift at a time. Both draw their noise from MUSAN and follow one shared mixing protocol, so the only thing that changes between them is the speech source. LSM (LibriSpeech-MUSAN) pairs read speech with out-of-family noise and isolates a noise-family shift. ESM (EARS-MUSAN) replaces the read speech with expressive EARS speech under the identical recipe and isolates a speech-style shift. Holding everything else fixed lets a single benchmark leg attribute a change in enhancement quality to one axis of the shift.

Construction protocol

Both sets are produced by one script and differ only in the speech source. For each pair the speech and noise are resampled to 16 kHz, the noise is looped to the speech length and offset by a seeded fraction, and its power is scaled to the target SNR. A single anti-clip rescale is applied to the clean and noisy signals together, so the pair stays perfectly aligned. Every random draw comes from one seeded generator (seed 1234 for LSM, seed 4321 for ESM), which makes the build bit-exact: regenerating from the public source corpora reproduces the released waveforms, and the published SHA-256 lists let you verify it.

Download & reproduce

Two ways to obtain the data. The lightweight route regenerates the exact waveforms from the public source corpora; the packaged route downloads the mixtures directly.

Regenerate from source (bit-exact)

  • Build scriptsbuild_lsm.pybuild_esm.py
  • Per-mixture manifestslsm_manifest.csvesm_manifest.csv
  • SHA-256 checksumslsm_sha256.txtesm_sha256.txt
  • Source corpora LibriSpeech MUSAN EARS

Download the source corpora, run each build script, and check the output against the SHA-256 lists. The manifests record the index, SNR, file names and per-clip checksums of the released set.

Packaged archives are attached to the repository release. Each archive holds clean/ and noisy/ at 16 kHz PCM16, named {lsm,esm}_INDEX_snrSNR.wav.

License & attribution

LSM is released under CC BY 4.0, inherited from LibriSpeech and MUSAN. ESM is released under CC BY-NC 4.0, inherited from EARS, and is therefore for non-commercial research use. When using either set, cite this work and the underlying source corpora (LibriSpeech, MUSAN, and, for ESM, EARS).
